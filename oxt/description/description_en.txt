LibreCalc AI Assistant

An intelligent AI-powered assistant for LibreOffice Calc that helps you:

- Read and analyze spreadsheet data
- Write formulas with natural language
- Apply cell styles and formatting (borders, colors, alignment)
- Detect and fix formula errors
- Get insights and summaries about your data
- Create charts and visualizations

Powered by OpenRouter, Google Gemini, and Ollama LLM providers.

Features:
- Natural language interface for spreadsheet operations
- 18+ spreadsheet tools with cell manipulation
- Google Gemini AI support
- OpenRouter multi-model support
- Dark/Light theme support
- Turkish/English bilingual interface
- Works offline with local Ollama instance
- Automatic update support

Requirements:
- LibreOffice 6.0 or later
- Python 3.6+
- Internet connection (for OpenRouter/Gemini) or local Ollama instance
